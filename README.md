# Blog

#### 介绍
基于SSM框架的个人博客

#### 软件架构
使用Spring4+Springmvc+Mybatis3架构；

采用Mysql数据库，数据库连接池使用的是阿里巴巴的Druid，使用Maven3管理项目；

前台网页使用主流的Bootstrap3 UI框架；后台管理使用主流易用的layUI轻量级框架；

在线编辑器使用了editor.md，支持图片上传，支持代码高亮特性；

服务器采用阿里云服务器，使用nginx代理，使用vsftpd搭建文件服务器。

留言板采用了云评论（原畅言）。
